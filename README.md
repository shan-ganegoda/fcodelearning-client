## Run Locally

Clone the project

```bash
  git clone https://github.com/shan-ganegoda/fcodelearning-client.git
```

Go to the project directory

```bash
  cd fcodelearning-client
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  ng serve
```
